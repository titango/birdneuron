export()

Export the current population models to a json file named “data.json”

exportBest()

export the model with the highest score to a json file named “data.json”

input(arrayObjects, useSavedData)

inputs models to population and active population

Parameters:

  • arrayObjects: an array of javascript Class objects
  • useSavedData: true will retain the current “brain” neuron of the object and false will create new “brain” neurons of the object
import(classParam)

inputs models to population and active population

Parameters:

  • classParam: Classname of the model
modelCopy(model)

copy the current model to a new object

Parameters:

  • Model: the current model
refreshPopulation()

Copy the models in ‘population’ array to the ‘active population’ array

Usage

Tutorial

CREATE Bird model
class Bird {
	      constructor()
	      {
			this.x = 64;
			this.y = 10;
	      }
	
	      copy() // Required!
	      {
			return new Bird();
	      }
		
		// function that returns an array of input numbers
		think()
		{
			. . . . .
		}
	
		// function that will determine what the bird will do 
// when receiving an output array
		do(outputs)
		{
			. . . . . .
		}

		// Display or draw the bird
		show()
		{
			. . . . . .
		}
}
*Note that function copy() is required which return new model object.


CREATE BirdNeuron object and insert models in main file.

//Using bird neuron
let bn = new BirdNeuron();

//Settings
bn.totalPopulation = 500;
bn.inputlayer = 5;
bn.hiddenlayer = 8;
bn.outputlayer = 2;

//Create bird models
      var tempBirds = [];
      for (let i = 0; i < bn.totalPopulation; i++) {
        let bird = new Bird();
        // console.log("bird: ", bird);
        tempBirds.push(bird);
      }
	
	//Insert to both population and total population
      bn.input(tempBirds, false);

	//Models in action - show the birds
for (let i = bn.activePopulation.length - 1; i >= 0; i--) {
          let bird = bn.activePopulation[i];

          // Bird uses its brain!
          bird.inputs = bird.think();

          var actions = bird.outputs();

          if(actions) bird.do(actions);
          bird.show();
}
